# conv.go 转换函数

## String2Int64
将字符串转换为int64类型

## Int642String
将int64类型转换为字符串。

## String2Int
将字符串转换为int类型。

## Int2String
将int类型转换为字符串。

## String2Uint
将字符串转换为uint类型。

## Uint2String
将uint类型转换为字符串。

## Float642String
将float64类型转换为字符串。

## String2Float64
将字符串转换为float64类型。
